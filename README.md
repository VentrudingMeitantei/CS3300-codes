# CS3300-codes
## MacroJava to MIPS Assembly Compiler
The above project is a amalgamation of 6 individual assignments done as a part of CS3300 Compiler Design Course. The individual parts can be described as:
* Syntax Analysis : Checks for Syntax errors in MarcoJava code and converts it into MiniJava after expanding macros.
* Type Checking : Checks for type-errors.
* IR generation 1 and 2 : Convert Minijava code to MiniIR and then later to MicroIR,
* Register Allocation : Generates MiniRA using linear scan to allocate registers to the temporaries generated in IR generation.
* MIPS generation : Converts MiniRA to MIPS Assembly code.

The grammars for MacroJava, MiniJava, MiniIR, MircoIR, MiniRA and MIPS assembly can be in the instructor Prof. Krishna Nandivada's webpage: https://cse.iitm.ac.in/~krishna/cs3300/subsets.html
